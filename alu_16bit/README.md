# 16-bit ALU (Verilog)

This project implements a 16-bit Arithmetic Logic Unit (ALU) using Verilog HDL.
The ALU supports arithmetic and logical operations with flag generation.

## Supported Operations
- ADD
- SUB
- AND
- OR
- XOR
- Shift Left
- Shift Right

## Flags Generated
- Zero
- Carry
- Sign
- Overflow

## Files
- `alu_16bit.v`      – ALU design module
- `alu_16bit_tb.v`   – Testbench for verification
